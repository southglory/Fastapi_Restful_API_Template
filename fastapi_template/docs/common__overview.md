## 인증 모듈

**경로**: `app/common/auth/`

JWT 토큰 기반 인증 및 비밀번호 해싱 기능을 제공합니다.

## 설정 모듈

**경로**: `app/common/config/`

애플리케이션 구성 설정을 관리하는 모듈입니다.

[설정 모듈 상세 문서](common_config.md)

**경로**: `app/common/database/`
